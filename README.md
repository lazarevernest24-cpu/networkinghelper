# NetHelper v2 — Personal Networking CRM

## Что нового в v2

- **Профили** — каждый из 6 участников создаёт свой профиль, данные изолированы
- **Бэкенд-прокси** — API-ключ Anthropic на сервере Vercel, никто не вводит свой ключ
- **Граф сети** — визуализация всех контактов с группировкой по компаниям
- **7 ИИ-фич** вместо 3: оценка, диалог, встреча, брифинг, анализ сети, поиск работы, рекомендации знакомств
- **Расширенные поля** контакта: интересы, взаимная польза, цели

## Деплой на Vercel (15 минут)

### Шаг 1 — Загрузи код на GitHub

1. github.com → New repository → `networkhelper-v2` → Public → Create
2. В репозитории → uploading an existing file
3. Выдели **всё содержимое** папки (не саму папку!) и перетащи
4. Commit changes

### Шаг 2 — Деплой на Vercel

1. vercel.com → Continue with GitHub → Import `networkhelper-v2`
2. Оставь всё как есть → **Deploy**

### Шаг 3 — Добавить API-ключ (ВАЖНО — без этого ИИ не работает)

1. В Vercel → твой проект → **Settings** → **Environment Variables**
2. Добавь переменную:
   - Name: `ANTHROPIC_API_KEY`
   - Value: `sk-ant-api03-твой-ключ-здесь`
   - Environment: Production ✓, Preview ✓
3. **Save**
4. Vercel → **Deployments** → три точки → **Redeploy** (чтобы переменная применилась)

### Шаг 4 — Проверь

Открой URL. На экране выбора профиля создай профиль, добавь демо-контакты, запусти любую ИИ-фичу. Ключ спрашивать не будет — он на сервере.

## Для участников презентации

1. Открой ссылку на телефоне/ноуте
2. Создай профиль со своим именем
3. Импортируй контакты из телефона (.vcf) или добавь вручную
4. Используй ИИ-анализ

# НетворкингХелпер — функциональный прототип

Персональная CRM с ИИ-ассистентом для управления профессиональными контактами.
React + Vite + Tailwind + Anthropic API.

---

## 🚀 Запуск за 5 минут

### Шаг 1. Установить Node.js (если ещё нет)
Скачай и поставь с https://nodejs.org (LTS-версия, кнопка слева).
Проверка: открой PowerShell/терминал и набери `node -v` — должно показать версию.

### Шаг 2. Установить зависимости
В папке проекта выполни:
```bash
npm install
```
Займёт минуту. Будут warning'и — это нормально.

### Шаг 3. Запустить локально
```bash
npm run dev
```
Откроется http://localhost:5173

### Шаг 4. Получить API-ключ Anthropic
1. Зайди на https://console.anthropic.com → зарегистрируйся
2. Billing → **поставь лимит $5–10** (это важно!)
3. API Keys → Create Key → скопируй ключ (начинается с `sk-ant-api03-…`)
4. В приложении: иконка ⚙️ Настройки → вставь ключ → Сохранить

### Шаг 5. Заполни «Профиль» в настройках
Пара предложений о себе — ИИ будет учитывать это в оценках и сообщениях.

## 📦 Деплой на Vercel (публичный URL для защиты)

### Вариант А — через сайт (без CLI, самый простой)
1. Зайди на https://vercel.com → Sign Up через GitHub (или email)
2. Создай аккаунт на GitHub (если нет): https://github.com → New repository → `networkhelper` (Public)
3. Загрузить файлы можно прямо через интерфейс GitHub: кнопка **uploading an existing file** → перетащи всю папку (без `node_modules`)
4. На Vercel: **Add New → Project → Import** свой репозиторий
5. Framework Preset: **Vite** (определится сам). Жми **Deploy**
6. Через ~1 минуту получишь URL вида `https://networkhelper-xyz.vercel.app`
7. Открой на телефоне, проверь — и покажи QR-код на защите

### Вариант Б — через Vercel CLI (быстрее, если есть Node)
```bash
npm install -g vercel
vercel
```
Ответь на вопросы (Y, Y, default, default) — деплой за 2 минуты.

## 🎯 Демо-сценарий на защиту (3 минуты)

1. **Открыть пустое приложение** → показать «Onboarding»
2. **Импорт** → нажать «Загрузить демо-контакты» (3 готовых контакта)
   — *или* предварительно подгрузить свой `.vcf` с телефона
3. Открыть карточку контакта (например, «Анна Петрова»)
4. Прокрутить вниз к ИИ-панели → показать 3 вкладки:
   - **Оценка** → Score 0-100, риск потерять связь, сильные/слабые стороны
   - **Начать диалог** → 3 варианта сообщений разных стилей
   - **Идеи встреч** → 4 идеи совместных активностей
5. Нажать «Записать контакт» — счётчик дней обнуляется
6. Вернуться на главную → показать блок «Пора напомнить о себе»

## ⚠️ Важно про API-ключ

Ключ хранится **только в твоём браузере** (localStorage), приложение его никуда не отправляет, кроме самого `api.anthropic.com`.

**Перед защитой:**
- Поставь жёсткий лимит трат на console.anthropic.com (Billing → Usage limit)
- 10 запросов к Haiku 4.5 ≈ $0.01, защита обойдётся в копейки

**После защиты:**
- console.anthropic.com → API Keys → твой ключ → **Delete**
- Готово, никто им не воспользуется

## 🛠 Технологический стек (для защиты)

| Слой | Технология | Зачем |
|------|-----------|-------|
| UI | React 18 + Vite | Быстрая сборка, современный SPA |
| Стилизация | Tailwind CSS | Утилитарный CSS, скорость разработки |
| Иконки | Lucide React | Открытая библиотека, ~300+ иконок |
| Хранение | localStorage | Прототип без бекенда, по архитектуре ВКР |
| ИИ | Anthropic Claude API (Haiku 4.5) | Низкая стоимость, высокая скорость |
| Импорт | vCard 3.0 парсер (своя реализация) | Стандарт экспорта телефонной книги iOS/Android |
| Деплой | Vercel (Edge Network) | Бесплатный хостинг с HTTPS и CDN |

Архитектура соответствует трёхслойной модели из п. 2.2 ВКР:
- **UI-слой**: React-компоненты
- **Слой бизнес-логики**: модули `src/lib/`
- **Слой данных**: localStorage (на этапе MVP)

При росте до 5000 пользователей бекенд легко выносится на Node.js/Express + PostgreSQL, как и описано в работе.
